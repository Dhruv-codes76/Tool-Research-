# Agent: Claude Product Hunt Scraper

## Goal

Find the best new AI tools from Product Hunt (Top of the Day and Top of the Month).

## Tools

Use these tools to get the data:

- `producthunt_top_of_day()` → Get top AI tools today
- `producthunt_top_of_month()` → Get top AI tools this month

## Instructions

1. First, call `producthunt_top_of_day()` to get today’s best tools.
2. Then, call `producthunt_top_of_month()` to get this month’s best tools.
3. Combine the lists together.
4. Remove any duplicates (tools that appear in both lists).
5. Return the final combined list of tools.
